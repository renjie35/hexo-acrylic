# CSS

Flex垂直主轴布局，固定高度 | 如何实现自适应宽度？

苏州

由于直接使用flex-direction: column;会导致高度坍塌

所以这边使用writing-mode: vertical-lr;来实现垂直布局, 由于writing-mode的垂直布局会导致子元素内容改变行进方向，所以这边使用writing-mode: horizontal-tb;来实现将子元素摆正
